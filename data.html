Data Sources Overview

Data Source Description Data Types Access Methods
PubChem A public repository of chemical information, including compound structures, properties, and bioactivity data. Compound structures, properties, bioassay data REST API, PUG, FTP, Bulk Data Download
ChEMBL A manually curated database of bioactive molecules with drug-like properties, including compound structures and bioactivity data. Compound structures, bioactivity data, target information Web Services, FTP, SQL Dump
DrugBank A comprehensive database of drug information, including drug targets, mechanisms of action, and pharmacokinetic data. Drug structures, targets, pharmacokinetic data, drug-drug interactions XML, CSV, SQL Dump
CTD The Comparative Toxicogenomics Database, which provides curated chemical-disease associations and chemical-gene interactions. Chemical-disease associations, chemical-gene interactions REST API, Downloads
SIDER A database of drug side effects, extracted from drug labels and public documents. Drug side effect information CSV files, MySQL dumps
