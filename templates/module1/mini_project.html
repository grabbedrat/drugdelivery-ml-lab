Mini-Project: Building a Molecular Similarity Search Tool

Introduction

In the field of drug discovery, identifying compounds with structural similarity to known active molecules can be a powerful strategy for finding potential drug candidates. This mini-project will challenge you to build a molecular similarity search tool using Python and the powerful RDKit library.

Your task is to develop a user-friendly application that allows researchers to input a reference molecule and search through a database of compounds to find those with similar structures. This tool can be invaluable in the early stages of drug discovery, enabling researchers to explore the chemical space efficiently and prioritize compounds for further evaluation.

Project Objectives

By completing this mini-project, you will:

  1. Gain hands-on experience with molecular representations and similarity calculations using RDKit.
  2. Implement an efficient similarity search algorithm to identify structurally similar compounds.
  3. Develop a user-friendly interface for inputting reference molecules and search parameters.
  4. Visualize and present the search results in a clear and informative manner.

Project Overview

  1. Data Preparation

    • Curate a database of molecules in SMILES format.
    • Preprocess the molecules and generate suitable representations (e.g., Morgan fingerprints).

  2. Similarity Search Algorithm

    • Implement a function to calculate the similarity between two molecules using the Tanimoto coefficient.
    • Develop a search algorithm to find the most similar compounds to a given reference molecule.

  3. User Interface

    • Create a command-line interface for users to input the reference molecule and search parameters.
    • Display the top-N most similar compounds along with their similarity scores.

  4. Evaluation and Refinement

    • Test the similarity search tool on a set of known similar compounds and assess its performance.
    • Refine the algorithm and user interface based on the evaluation results.

Getting Started

To get started with the mini-project, follow these steps:

  1. Install the required dependencies, including Python and the RDKit library.
  2. Download the starter code and dataset from the provided link: Mini-Project Notebook
  3. Open the Jupyter notebook and complete the tasks outlined in each section.
  4. Once you have completed the notebook, combine the code into a single Python script and test the final similarity search tool.

Submission Guidelines

To submit your mini-project, follow these guidelines:

  • Combine the code from the Jupyter notebook into a single Python script.
  • Include a README file with instructions on how to run the similarity search tool and any dependencies required.
  • Create a ZIP file containing the Python script, README file, and any additional files or datasets used.
  • Upload the ZIP file to the designated submission portal or email it to the instructor.

Good luck with the mini-project! If you have any questions or need assistance, don't hesitate to reach out to the instructor or your peers.
