Module 3: Molecular Representation and Featurization

Welcome to Module 3 of the Flask Computational Chemistry ML Lab for Drug Discovery! In this module, you'll dive into the world of molecular representation and featurization techniques, which are essential for preparing chemical data for machine learning models.

Module Overview

  1. Introduction to Molecular Representation
  2. SMILES Strings and RDKit
  3. Extended-Connectivity Fingerprints (ECFPs)
  4. Molecular Descriptors
  5. Graph-Based Representations
  6. Molecular Embeddings (Mol2vec)
  7. Hands-on Exercises and Quizzes
  8. Real-world Applications and Case Studies
  9. Discussion and Reflection

Getting Started

To get started with Module 3, you have two options:

  1. Google Colab: Click the button below to open the Module 3 notebook in Google Colab:
    Open Module 3 Notebook in Google Colab
  2. Local Jupyter Notebook: If you prefer to run the notebook locally, follow these steps:
    1. Install the necessary dependencies and libraries by running pip install -r requirements.txt.
    2. Launch Jupyter Notebook by running jupyter notebook in your terminal.
    3. Navigate to the module directory and open the module3.ipynb notebook.
